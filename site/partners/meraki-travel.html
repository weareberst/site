Drawing on local expertise, they offered destination inspiration and all the information people need to create and book their own unforgettable holidays. But their site was cumbersome and frustrating to use, especially on smaller screens - instead of being a delightful adventure, it was letting users down and holding Meraki back. {% endset %} {% component "content/text", { text: text } %} {% component "content/image-pair", { images: [ { src: "https://placekitten.com/g/600/400", caption: "Research and journey mapping revealed the most promising opportunities" }, { src: "https://placekitten.com/g/900/500", caption: "Another image with a caption" } ] } %} {% set text %} ## Exploring new ground Over three two-weeks sprints at Meraki’s offices in London, we worked with the team to completely redesign their adventure-booking experience based on research, prototyping and validation with real customers. Our priorities were to bring the Meraki difference to the surface - their unique real-time adventure-builder - and to create a simpler, more intuitive and responsive trip-building tool that people would love using on any screen. To this end, we created new product experience principles, information architecture and completely redesigned visual language. {% endset %} {% component "content/text", { text: text } %} {% component "content/feature-image", { small: "/img/temp-meraki-2.jpg", medium: "/img/temp-meraki-2.jpg", large: "/img/temp-meraki-2.jpg", caption: "Optional caption goes here" } %} {% set text %} ## The difference we made The new booking experience made using Meraki playful, intuitive and a joy for would-be travellers - an adventure in itself.
